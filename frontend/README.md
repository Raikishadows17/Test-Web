# TLS System Web

Requirements:
Node Js 24.11.1 (LTS)
AngularCLI (20.3.9)

##Instalacion
Instalar angular desde la terminal. Por defecto se instala la ultima version de Angular Cli 20.3.9

##
```
npm install -g @angular/cli
```

##Para correr el proyecto se debe instalar las dependencias.

```
npm install
```


## Development server

To start a local development server, run:

```bash
ng serve -o
```
